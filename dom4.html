DOM adalah singkatan dari Document Object Model. DOM JavaScript adalah interface yang memungkinkan developer memanipulasi halaman web dari segi struktur, tampilan, dan kontennya

Event DOM di JavaScript adalah jenis event yang terjadi pada elemen-elemen DOM (Document Object Model) di halaman web. Ini berarti, ketika pengguna berinteraksi dengan elemen-elemen HTML seperti tombol, input, atau elemen lainnya, atau ketika halaman web dimuat, event-event ini dipicu. Berikut adalah beberapa contoh event DOM umum:
onclick: Terjadi saat pengguna mengklik sebuah elemen.

Javascript DOM Event dan Event Listener
Event Listener adalah mekanisme di JavaScript yang memungkinkan kita untuk "mendengarkan" atau "menyimak" suatu event tertentu pada elemen DOM, dan menentukan tindakan atau fungsi yang harus dijalankan saat event tersebut terjadi. c contoh yaitu pada saat button di click (button onclick="namafungsi())
