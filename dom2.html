DOM adalah singkatan dari Document Object Model. DOM JavaScript adalah interface yang memungkinkan developer memanipulasi halaman web dari segi struktur, tampilan, dan kontennya

- mengubah element html dan properti propertinya beserta css
Apa saja yang bisa kita ubah dengan javascript? :
1. document.getElementById().innerHTML (mengubah konten dari elemen html)
2. document.getElementById().attribute (mengubah attribute elemen html)
3. document.getElementById().style.property (mengubah atau menambah css style pada elemen html)
