DOM adalah singkatan dari Document Object Model. DOM JavaScript adalah interface yang memungkinkan developer memanipulasi halaman web dari segi struktur, tampilan, dan kontennya

- untuk melakukan dom dengan hmtl kalian dapat mengakses dengan cara :
1. document.getElementById (mengakses elemen html dari id nya)
2. document.getElementsByTagName (mengakses elemen html dari tag)
3. document.getElementsByClassName (mengakses elemen html dari nama class)
